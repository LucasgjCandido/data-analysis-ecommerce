# Análise de Dados de Vendas de E-commerce

Este projeto tem como objetivo analisar dados de vendas de um e-commerce para gerar indicadores de desempenho e insights de negócio.

## Análises realizadas

- Faturamento total
- Ticket médio
- Ranking de produtos mais vendidos
- Vendas por mês
- Distribuição de vendas por categoria

## Tecnologias utilizadas

- Python
- Pandas
- SQL
- Excel

## Objetivo

Demonstrar habilidades em análise de dados, manipulação de datasets e geração de métricas para apoio à tomada de decisão.

## Estrutura do projeto

dataset.csv → base de dados utilizada  
analysis.py → script de análise de dados  
dashboard → visualizações e indicadores
